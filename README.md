# My Portfolio

A modern and responsive personal portfolio website showcasing my skills, projects, experience, and professional journey as a software developer.

## Overview

This portfolio serves as my online presence where visitors can learn more about me, explore my projects, view my technical skills, and get in touch.

The website is designed to be clean, responsive, and user-friendly across desktop and mobile devices.

## Features

- Responsive design for all screen sizes
- Professional introduction section
- Skills showcase
- Project portfolio section
- Contact information
- Clean and modern UI
- Fast loading static website

## Technologies Used

- HTML5
- CSS3
- Responsive Web Design

## Getting Started

### Prerequisites

You only need a modern web browser.

### Installation

1. Clone the repository

```bash
git clone https://github.com/Maybach-Zee/MyPortfolio.git
```

2. Navigate to the project folder

```bash
cd MyPortfolio
```

3. Open the project

Simply open `index.html` in your browser.

## Running Locally

You can run the project using VS Code Live Server:

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
